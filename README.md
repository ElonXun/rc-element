# rc-element
>该开源组件库基于 [ElementUI库](http://element-cn.eleme.io/#/zh-CN/component/progress) 的组件样式
将Element组件用React配合TypeScript进行重写
开发过程中部分组件参考[AntDesign](https://ant.design/docs/react/introduce-cn)组件的写法
