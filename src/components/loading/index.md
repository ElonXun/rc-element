## API

### Tag Attributes

| 参数 | 说明 | 类型 | 可选值| 默认值 |
| --- | --- | --- | --- | --- |
| type | 标签主题 | string | success/info/warning/danger | - |
| closable | 标签是否可以关闭 | boolean | - | false |
| tagKey | 标签标示 | string | - | - |
| size | 标签尺寸 | string | medium / small / mini | - |
| style | 自定义style | object | - | - |
